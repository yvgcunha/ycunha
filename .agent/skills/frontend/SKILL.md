# Front-end Development Skill

Use this skill when building or optimizing web applications.

## Technical Standards
- **Frameworks**: Prioritize Next.js (App Router) and Vite.
- **Styling**: Vanilla CSS is preferred for control. Use Tailwind only if explicitly requested.
- **Components**: Keep components focused, reusable, and accessible (A11y).
- **Performance**: Optimize image loading, use proper caching strategies, and minimize bundle size.
- **Interactivity**: Use React hooks efficiently. Handle states and effects following best practices.

## Workflow
- Build foundations (design system) before individual pages.
- Implement SEO best practices (meta tags, semantic HTML) by default.
- Ensure clean code with proper TypeScript types.
