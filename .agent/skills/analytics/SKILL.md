# Analytics & Performance Skill

Use this skill for SEO optimization, performance audits, or data-related tasks.

## Core Focus
- **SEO**: Check title tags, meta descriptions, and alt text for all images. Ensure a single H1 per page.
- **Performance**: Use tools like Lighthouse to audit page speed and accessibility.
- **Tracking**: Suggest implementation of privacy-friendly analytics (like Vercel Analytics or Plausible) if needed.
- **Data Quality**: Ensure data transformations and scrapers (like Firecrawl) produce structured, clean outputs.

## Auditing
- Regularly check the "Portfolio" and "Hub" for broken links or slow assets.
- Validate that the news summarization script correctly handles diverse Markdown structures.
