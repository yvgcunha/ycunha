# Design Skill

Use this skill when the task involves UI creation, layout design, or aesthetic improvements. Always aim for a "WOW" factor.

## Core Principles
- **Modern Aesthetics**: Favor glassmorphism, subtle gradients, and dark modes.
- **Typography**: Use Google Fonts (Inter, Roboto, Outfit) instead of system defaults. Ensure proper hierarchy.
- **Color Theory**: Avoid generic colors. Use curated HSL palettes.
- **Micro-animations**: Implement hover effects and smooth transitions to make the UI feel alive.
- **Spacing**: Use generous white space to prevent cluttered layouts.

## Implementation Guidelines
- Start with `index.css` for design system tokens.
- Use CSS variables for colors, spacing, and shadows.
- Ensure 100% responsiveness without compromising visual quality.
